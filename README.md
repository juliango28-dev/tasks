# GESTOR DE TAREAS
CRUD elaborado en PHP, el cual permite asignar un usuario responsable activo a una tarea, las tareas pueden ser modificadas y reasignadas a otros usuarios, cada tarea cuenta con un estado de progreso y una prioridad.  

## Tecnologias utilizadas
- Base de datos (MariaDB)
- PHP 
- HTML 
- CSS básico 
- jQuery.
